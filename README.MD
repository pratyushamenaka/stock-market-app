# Stock Market App

This is a Backend API for forex-ui front end react app to display currencies and calculate best profit made by buying and selling currency at any given day.
The data is processed using JPA and Hibernate and loaded into Oracle database.

It also has an example model, controller, database schema, input data sql, repository, and service files.


## Getting Started


To clean build  and run the stock-market-app, use below commands:

```
./gradlew clean build 
```

```
./gradlew bootRun
```

### Test Coverage
Test Coverage is done using Groovy with Spock.
